🔍 Monitor de Redes Gamer & Pericial

Este projeto foi desenvolvido para monitorar a estabilidade de conexões de rede em tempo real, sendo uma ferramenta útil para diagnosticar lag em jogos e para a coleta de logs em contextos de perícia digital.

🛠️ Funcionalidades
- Teste de Ping Automatizado: Verifica a comunicação com servidores (como Google ou servidores de jogos).
- Logs de Auditoria: Gera automaticamente um ficheiro `log_rede.txt` com o estado da conexão e carimbo de data/hora.
- Interface Simples: Mostra no terminal se o servidor está "Ativo" ou "Inacessível".
- Tratamento de Interrupção: Permite encerrar o programa de forma limpa usando `Ctrl + C`.

🚀 Como Utilizar
1. Certifica-te de que tens o Python 3 instalado.
2. Faz o download do ficheiro `rastreador.py`.
3. Executa no terminal: `python3 rastreador.py`.

💻 Tecnologias
- Python (Bibliotecas `os` e `time`).
- Linux Terminal (Comando `ping`).
